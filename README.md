# The Analysis Lab

A classroom web resource built for high school English students at Concordia International School Shanghai.

## What it covers

- **8 analysis skills** — claims, subclaims, counterclaims, textual evidence, elaboration, evaluation of evidence, synthesis, and conclusions with implications
- **Interactive lessons** — a common literary passage used across all eight skills, with weak examples, strong examples, and reflection prompts
- **AI-powered practice tool** — students submit their own writing for feedback; the coach gives one strength and two areas for improvement but will not write for them
- **MLA & citation** — in-text citation rules, Works Cited entry types, block quotation formatting, and a direct link to Purdue OWL
- **MLA coach** — a narrowly scoped AI tool that answers citation questions only

## How to use it

Open the site in any browser. No login required. No data is collected or stored.

## Built with

HTML, CSS, and vanilla JavaScript. AI feedback powered by the Anthropic API (Claude).

## For teachers

The AI coach is designed with hard guardrails — it will not write essays, complete sentences, or produce analytical content on a student's behalf. It identifies strengths and areas for improvement only.
